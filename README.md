\# Sinema Bilet Rezervasyon Sistemi



Bu proje, kullanıcıların sinema filmlerini ve seanslarını görüntüleyerek

koltuk seçimi yapabildiği ve bilet rezervasyonu oluşturabildiği

console tabanlı bir Java uygulamasıdır.



\## Özellikler

\- Film ve seans listeleme

\- Koltuk seçimi

\- Rezervasyon oluşturma

\- Rezervasyon geçmişini görüntüleme

\- İlk 3 bilet için %20 indirim uygulaması



\## Kullanılan Teknolojiler

\- Java

\- Eclipse IDE

\- Git \& GitHub

\- JUnit



\## Nesne Tabanlı Programlama

\- Encapsulation

\- Inheritance

\- Polymorphism

\- Interface kullanımı



\## Çalıştırma

Projeyi Eclipse üzerinden çalıştırarak console ekranından kullanabilirsiniz.
