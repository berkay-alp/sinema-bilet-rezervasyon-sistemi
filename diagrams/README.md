\# UML Diyagramları



\## Sınıf Diyagramı

Sınıf diyagramı, sistemde yer alan sınıfları ve bu sınıflar arasındaki

ilişkileri göstermektedir. Kalıtım, interface ve nesneler arası ilişkiler

UML standartlarına uygun olarak modellenmiştir.



\## Use Case Diyagramı

Use case diyagramı, kullanıcının sistem ile olan etkileşimini ve

rezervasyon sürecindeki adımları detaylı şekilde göstermektedir.
